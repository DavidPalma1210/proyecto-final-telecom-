# Proyecto Final: Telecomunicaciones

Este repositorio contiene la descomposición del proyecto final de telecomunicaciones para identificar operadores ineficaces en el servicio CallMeMaybe.

## Objetivo

Analizar datos de llamadas para detectar operadores con posibles señales de bajo desempeño, considerando llamadas entrantes perdidas, tiempo de espera y actividad saliente.

## Contenido

- `proyecto_final_descomposicion.ipynb`: plan de trabajo en formato Jupyter Notebook.
- `data/`: datasets utilizados para el proyecto.

## Estado del proyecto

Etapa actual: descomposición y planificación del análisis.
